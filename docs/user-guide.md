## 1. Introduction
Inovexa-AI est un ERP nouvelle génération avec intelligence artificielle intégrée.

## 2. Première connexion
1. Ouvrez votre navigateur
2. Allez à l'URL de votre application
3. Cliquez sur "Register"
4. Remplissez le formulaire
5. Connectez-vous avec vos identifiants

## 3. Dashboard
Le dashboard affiche les indicateurs clés :
- Chiffre d'affaires
- Produits en stock
- Nombre d'employés
- Taux de conversion

## 4. Gestion des factures
- Créer une facture : cliquez sur "Nouvelle facture"
- Modifier : cliquez sur l'icône d'édition
- Supprimer : cliquez sur l'icône corbeille
- Exporter : cliquez sur "Excel" ou "PDF"

## 5. Gestion des produits
- Ajouter un produit : remplissez le formulaire
- Modifier le prix : modifiez les champs
- Gérer les stocks : suivez les alertes

## 6. Gestion des employés
- Ajouter un employé : saisissez les informations
- Gérer les congés : créez une demande
- Paie : consultez les bulletins

## 7. Gestion des clients
- Ajouter un client : informations de contact
- Limite de crédit : définissez un montant
- Historique : consultez les commandes

## 8. Production
- Créer une nomenclature (BOM)
- Lancer un ordre de fabrication
- Contrôle qualité

## 9. Logistique
- Créer une expédition
- Suivre les livraisons
- Gérer les transporteurs

## 10. Assistant IA
- Posez vos questions
- Obtenez des recommandations
- Prédictions de ventes

## 11. Rapports
- Rapports financiers
- Rapports de stock
- Rapports RH
- Rapports ventes

## 12. Paramètres
- Profil utilisateur
- Changer mot de passe
- 2FA (double authentification)
- Notifications

## 13. FAQ
**Q: Comment réinitialiser mon mot de passe ?**
R: Cliquez sur "Mot de passe oublié" sur la page de connexion.

**Q: Comment exporter mes données ?**
R: Utilisez les boutons "Excel" ou "PDF" dans chaque module.

**Q: L'assistant IA ne répond pas ?**
R: Vérifiez votre connexion internet et rechargez la page.
